Bachelorarbeit Julia Bengelsdorf

## Vorläufiges Studiendesign (2x2 Design, 1x3 Design oder 1x2 Design)
### Abhängige Variable
+ Engagement 
  (Operationalisierung: Akkuratesse der Schätzungen, 
  verwendete Zeit/Aufgabe, Vollständigkeit der Fragebogenbearbeitung, 
  Screener)

## 2x2 Design
### Unabhängige Variablen

+ Disclaimer (Warnung) -  "Bitte bearbeiten Sie diese Studie sorgfältig.
  Die Sorgfältigkeit der Bearbeitung wird im Rahmen dieser Studie 
  kontrolliert." 
  (Between-Subjects-Manipulation: Disclaimer / kein Disclaimer)
+ Screener - z.B. "Geben Sie bei der folgenden Schätzung bitte "0" an."
  (Between-Subjects-Manipulation: Screener / keine Screener)

### Versuchsbedingungen
+ 1: Disclaimer + keine Screener
+ 2: kein Disclaimer + keine Screener
+ 3: Disclaimer + Screener
+ 4: kein Disclaimer + Screener

## 1x3 Design
### Unabhängige Variable:
+ Engagement-Förderung
    * Problem: Für ein sauberes Versuchsdesign würde noch die
    Versuchsbedingung Disclaimer + keine Screener fehlen
      
### Versuchsbedingungen
+ 1: kein Disclaimer + keine Screener
+ 2: kein Disclaimer + Screener
+ 3: Disclaimer + Screener
      

## 1x2 Design
### Unabhängige Variable
+ Engagement-Förderung
  * Problem: Konfundierung von Disclaimer und Screener

### Versuchsbedingungen
+ 1: Maximale Engagementbedingung (Disclaimer + Screener)
+ 2: Keine zusätzliche Engagement-Förderung 
  (kein Disclaimer + keine Screener)
