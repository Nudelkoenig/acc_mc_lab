# exp_acc_mc_lab
Laborexperiment Manipulationskontrolle der Interviewmanipulation von Accountability

## Studiendesign (1x3 Design)
### Abhängige Variable
+ Instruktonskontrolle
+ Manipulationskontrollen (gefühlte outcome und process accountability)
+ Psychologische Mechanismen (epistemische Motivation und Stress)
+ Akkuratheit (MAPE)

### Unabhängige Variablen

+ Accountability
  (Operationalisierung: Outcome accountability, process accountability, no accountability, via Interviewmanipulation)
